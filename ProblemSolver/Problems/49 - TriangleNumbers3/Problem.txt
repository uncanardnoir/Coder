# Problem Description
Recall from Problem 47 that the $n^{th}$ triangle number is defined as $1 + 2 + 3 + ... + n$. Given an array of numbers, return true if every number in the array is a triangle number, and false otherwise.
%%%
# Input Parameter
System.Int32[]
# Max input size
0
# Output Parameter
System.Boolean
# Sample Input
[15, 6, 3]
# Sample Output
true
