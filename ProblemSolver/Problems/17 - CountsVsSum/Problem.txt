# Problem Description
Given an array of up to 2000 integers (which can be zero or negative), return true if the sum of all integers is greater than the number of integers, and false otherwise. For instance, with the array $[1, 3, -5, 0, -1]$, the sum of all integers is -2 and the number of integers is 5, so the output would be false.
%%%
# Input Parameter
System.Int32[]
# Max input size
2000
# Output Parameter
System.Boolean
# Sample Input
[1, 3, -5, 0, -1]
# Sample Output
false
